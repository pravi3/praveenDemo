$Apply

Used as: $scope.$apply()
If code is not a part of angular js context angular can't able observe changes happened on code. Example:
  • if Jquery is used for handling click event then changes happen inside click function not observed angular js, even angular code present inside function angular unable to observe changes
  • Simller to above example if onClick function used inside html angular js doesn't observe changes
  • But instead of above two approach if we use ng-click ,changes are observed by angular js

    • By looking into above example when click event called using ng-click directive changes are observed because all built in directive by default calls $apply function after execution of defined function of angularjs
    When used manually calling $apply means user is trigging all watch function created on application (include angular created and user created) means angular calls all watchers starts from $rootscope followed by child scope object($scope) this event make angular to update with latest value on entire application.
