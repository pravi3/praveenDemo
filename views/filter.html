Number and Currency Filter Example
Number

Input data 1234567787.22000555000

Filter out is {{1234567787.22000555000|number}}

Number: Passing argument to Filter

Input data 1234567787.22000555000

Filter out is {{1234567787.22000555000|number:2}}

Currency

Input data 1234567787.22

Filter out is {{1234567787.22|currency}}

Currency: Passing argument to Filter

Input data 1234567787.22333 and passing argument as '@'

Filter out is {{1234567787.22333|currency:'@':2}}

Date Filter: Passing Argument to Date Filter

Input data 1288323623006

Custom filter Example
Filter Capatalizes Mentioned Character of word. if length doesn't matches it just returns input as it is.
=====> example : abcd:2 : {{'abcd'|charuppercase:2}}
=====> abcd:5 : {{'abcd'|charuppercase:5}}
